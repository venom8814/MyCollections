## Инструкция по запуску

1. **Скачайте архив проекта** и распакуйте его на своем компьютере.
2. Откройте Android Studio.
3. В меню **File → Open** выберите папку с распакованным проектом.
4. Дождитесь полной синхронизации Gradle.
5. Перед запуском убедитесь, что у вас установлен JDK 11 или 17, а также выбран эмулятор или подключено устройство.
6. Нажмите **Run** (зеленая стрелка) для запуска приложения.
7. В приложении вы сможете:
   - Просматривать список элементов коллекции
   - Добавлять новые элементы через окно "Добавить"
   - Удалять элементы, нажав кнопку "Удалить" рядом с ними
